# AI Sales & Inventory Analytics Bot

Telegram-бот для автоматизированного мониторинга продаж и глубокой бизнес-аналитики. Система позволяет владельцам бизнеса получать ключевые показатели (KPI) и интеллектуальные рекомендации на основе данных из CRM в режиме реального времени.

## Основные возможности

*   **Моментальные отчеты:** Получение данных по ключевым метрикам за любой выбранный период.
*   **Интерактивное управление:** Удобная навигация через Telegram-интерфейс (быстрые кнопки).
*   **Гибкий выбор дат:** Встроенный WebApp для выбора произвольного диапазона дат.
*   **AI-аналитика:** Автоматическая интерпретация данных от ИИ с конкретными рекомендациями по развитию бизнеса.

## Интерфейс пользователя

Управление ботом интуитивно понятно и не требует ввода сложных команд.

1. **Панель управления (Quick Picks):**
   Пользователь может быстро выбрать стандартный период (Сегодня, Неделя, Месяц) через главное меню.

2. **Выбор произвольной даты:**
   Для детального анализа за специфический период бот открывает удобный WebApp календарь.

## Пример отчета

Бот формирует структурированный ответ с KPI и аналитическими советами:


KPI:
- Унікальних товарів: 1
- Загальний прибуток: 3850.00 грн
- Загальна виручка: 9600.00 грн
- Маржа: 40.10%

ТОП ТОВАРІВ (назва - прибуток):
*Ваш товар* - 3850 грн

РЕКОМЕНДАЦІЇ:
1. Оскільки весь прибуток отримано від *вашого товару*, варто розширити асортимент, щоб знизити ризики залежності від окремого продукту.
2. При маржі 40% є простір для підвищення цін або оптимізації витрат, що дозволить збільшити прибуток без значного зростання обсягів продажу.
